│     - Требуется Android Studio & SDK                            │
│                                                                │
│  3️⃣  Через Android Studio:                                     │
│     - Откройте папку android/                                  │
│     - Build → Build APK(s)                                     │
│     - APK будет в app/build/outputs/apk/debug/                  │
